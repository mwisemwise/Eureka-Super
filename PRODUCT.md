# Product

## Positioning
Eureka Super is a calm, capture-first idea tool that keeps privacy local and retrieval fast.

## ADHD-friendly design principles
- Default to one clear action: capture now
- Keep high-frequency actions visible
- Hide advanced features until needed
- Reduce visual noise and mode confusion

## Core user flows
1. Capture: open app, type idea, save in seconds
2. Retrieve: search by title/content, sort quickly
3. Secure backup: export plain or encrypted JSON

## MVP vs future features
- MVP
  - fast capture
  - ideas list with search/sort
  - optional stats
  - help/onboarding
  - encrypted export
- Future
  - import and merge conflict handling
  - richer voice capture pipeline
  - resurface scheduling and reminders

## Resurface (future flagship)
Resurface will intelligently bring back older ideas based on recency gaps, tags, and past activity so good ideas re-enter active thinking without manual digging.
